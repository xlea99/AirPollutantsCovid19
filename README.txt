Data Visualization - Project 2 - Alex Somheil, Z1888439
=======================================================

This data visualization tool seeks to display information concerning the potential effects that COVID-19, and subsequent lockdown periods,
had on pollutant levels across several major, distinct US cities. This solution is displayed as a webpage. This site can be readily accessed
at the following link:

https://students.cs.niu.edu/~z1888439/DataVisualization/project2/

For local hosting, use the given project files and deploy them in a web server environment. The index.html file 
should be placed in a directory that is served by a web server, and the given file hierarchy should be maintained.
